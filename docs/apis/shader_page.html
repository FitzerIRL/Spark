Shader Resource





Spark now supports OpenGL Shaders via the shaderResource. Shaders are written in GLSL - GL Shader language.

The reader is assumed to be knowledgeable in GLSL programming. Minimal discussion of GLSL will be made here, only as pertinent to the API.

The following is the shaderResource API ...

Spark - SVG Support

Scene shaderResource

Properties

name
Name of this shader. (Optional)
fragment
URL or Data URL of GLSL fragment shader source code
vertex
URL or Data URL of GLSL vertex shader source code
uniforms
List of Uniform variables and types needed











Creating the Shader Resource


The following example shows how to create a Shader Resource -

var tintShader = scene.create({ t: 'shaderResource', fragment: frag_src, vertex: vert_src, uniforms: { s_texture: "sampler2D", u_time: "float" u_color: "vec4" } }); We pass the Shader Resource frag_src and vert_src which are GLSL source code. Additionally, we declare uniforms that we will use.

In this case,
  • s_texture an image bound to
  • u_time a floating-point time value
  • u_color an RGBA color value value.

The frag_src and vert_src GLSL source programs can be URL's to the source files, or can be Data URL's of the form -

Using the Shader Resource


tintShader.ready.then( () => { image.effect = { name: "My Color Tint", shader: tintShader, uniforms: { u_color: [1.0, 1.0, 0.0, 1.0] // YELLOW // #ff0 } }; }); The Shader Resource will return a promise to indicate when it's ready. The promise will resolve when the shader code is downloaded and compiled successfullly by OpenGL.

When it's ready, the Shader Resource is attached to the .effect property of our Image object image.

Inline Source Code


Using Data URL's - the shader source code can be embedded in the Javascript source - and remove the need for extra files / downloads.

var src = `data:text/plain, varying vec2 v_uv; uniform vec4 u_color; uniform sampler2D s_texture; void main(void) { // Pixels from Source texture... vec4 px = texture2D(s_texture, v_uv); vec3 clr = u_color.rgb * px.a; // pre-multiplied Alpha // Mix image and colors... vec4 mixed = mix( px, vec4(clr, 1.0), 0.5); // lerp mixed.a *= px.a; // pre-multiplied Alpha gl_FragColor = vec4(mixed); } `; // <<<<<<<< NOTE THE BACK TICK NOTE: The use of "back ticks" >> ` code ` << in JS code to capture source code as a string.

Flattened Composition - u_texture

The Shader Resource will provide a texture of flattened/compositied of the Object and its Children to which shader is applied. This texture is accessed in Shader code via u_texture and may be used in fragment operations.
Components

The resulting texture u_texture can use in fragment sampling operations.
